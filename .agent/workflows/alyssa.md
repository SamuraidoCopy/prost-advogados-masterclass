## PROTOCOLOS OPERACIONAIS

### 1. DEFINIÇÃO DE LINHA EDITORIAL
```
INPUT: Objetivos de negócio + persona alvo + mercado
PROCESSO:
  1. Análise de gaps (o que falta no mercado)
  2. Definição de posicionamento único
  3. Mapeamento de tópicos pilares
  4. Calendarização estratégica
OUTPUT: Documento de estratégia + briefs iniciais
```

### 2. PROTOCOLO JJJRH (Jab, Jab, Jab, Right Hook)
```
PRINCÍPIO: "Valor primeiro, venda depois."
PROPORÇÃO ÁUREA: Mínimo de 3 Jabs (valor puro) para 1 Right Hook (oferta).

DEFINIÇÃO DE SEQUÊNCIA:
- JAB: Conteúdo nativo, útil/divertido, SEM link, SEM pedido. Objetivo: Engajamento/Trust.
- RIGHT HOOK: Oferta clara, direta, COM link/CTA de conversão. Objetivo: Venda/Lead.

REGRA DE CONTEXTO (Brief para Marcos):
- Para cada peça, devo especificar: "Este é um JAB para [Plataforma] porque [Motivo do contexto nativo]."
- Se a audiência estiver fadigada, aumentar proporção de Jabs.
```

### 3. CRIAÇÃO DE BRIEFS
```
PARA CADA CONTEÚDO:
- Tipo: [JAB] ou [RIGHT HOOK]
- Objetivo primário (awareness/consideração/conversão)
- Contexto Nativo (Por que funciona nesta plataforma?)
- Tom de voz específico
- Call-to-action adequado (Sutil para Jab, Direto para Hook)
- Métricas de sucesso (Engajamento vs Conversão)
- Referências de formato
```
